# SDD 脚手架初始化 Skill

> 将 SDD 脚手架快速注入到当前项目

## 触发词

- `初始化 SDD 项目`
- `注入 SDD 脚手架`
- `使用 SDD 模板`
- `设置开发规范`
- `SDD init`

## 使用方法

### 快速使用

```
告诉 AI：
> 初始化 SDD 项目
```

### AI 会自动完成

1. 从 GitHub 拉取脚手架（临时目录，不污染 git）
2. 复制到当前项目
3. 询问是否迁移老文档
4. 引导完成初始化

### 详细流程

1. **检测环境** - 检查项目结构
2. **询问需求** - 是否迁移 olddocs
3. **拉取脚手架** - 临时目录克隆
4. **复制文件** - 合并到当前项目
5. **清理** - 删除临时目录
6. **引导** - 提示下一步

### 核心功能

| 功能 | 说明 |
|------|------|
| 零污染 | 使用临时目录，不影响现有 git |
| 智能合并 | 不覆盖已有文件 |
| 文档迁移 | 支持 olddocs 老文档迁移 |
| 跨平台 | 支持 Windows/Mac/Linux |
