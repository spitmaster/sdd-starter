# 旧文档示例 - 需求文档

## 产品需求文档 (PRD)

### 项目名称
XX管理系统

### 核心功能
1. 用户管理 - 注册、登录、权限控制
2. 内容管理 - CRUD 操作
3. 数据分析 - 报表统计

### 目标用户
- 管理员：系统配置和用户管理
- 普通用户：日常业务操作

### 优先级
| 功能 | 优先级 | 预计工时 |
|------|--------|---------|
| 用户认证 | P0 | 3天 |
| 内容管理 | P1 | 5天 |
| 数据报表 | P2 | 4天 |

### 约束条件
- 需要支持移动端访问
- 数据量预估 100万+ 级别
- 响应时间 < 200ms

## 用户故事

### US-001: 用户注册
**作为** 访客
**我想要** 注册账号
**以便** 使用系统功能

**验收标准：**
- 可以输入邮箱和密码
- 密码长度 >= 8位
- 注册成功后自动登录

### US-002: 内容搜索
**作为** 普通用户
**我想要** 搜索内容
**以便** 快速找到所需信息

**验收标准：**
- 支持关键词搜索
- 返回结果 < 1秒
- 支持分页浏览
